Quick Installation

Follow these simple steps to get started:

  1. Install the module from Odoo Apps
  2. Configure diesel product and operation type (optional)
  3. Start creating diesel logs for your fleet

Configuration

Navigate to Settings > Diesel Log Configuration to configure:

  • Diesel Product: Product used for fuel
  • Fuel Operation Type: Stock operation for fuel issues

If not configured, the system automatically creates:

Product: "Diesel" (Liter UoM) Operation Type: "Fuel Issue" - Source: "Fuel Tank" - Destination: "Fuel Consumption"
