🎯 KEY CAPABILITIES
───────────────────────────────────────────────────────────────────

1. CONTRACT LIFECYCLE
   Draft → Offer → Negotiation → Sign Pending → Active → Suspended/Expired
   - Odoo Sign integration (mandatory)
   - Dynamic web agreement portal
   - Auto-activation on signature

2. CONTRACT TYPES
   ┌──────────────────┬─────────────────┬──────────────────┐
   │ driver_transport │ Diesel (req)    │ Maintenance (opt)│
   │ pump_ops         │ Maintenance (req│ Diesel (opt)     │
   │ accounts_audit   │ Attendance (req)│ Diesel/Maint(opt)│
   └──────────────────┴─────────────────┴──────────────────┘

3. WAGE STRUCTURE
   Part-A (Fixed)    : Base manpower cost (designation × headcount × rate)
   Part-B (Variable) : Linked to MGQ achievement %
   Deductions        : Clause 9 breakdowns
   Bonus/Penalty     : Star rating adjustments
   Material Variance : Inventory shortage/excess
   TDS 194C          : Auto @ 2%

4. PERFORMANCE SYSTEM
   Score = Diesel(50%) + Maintenance(30%) + Attendance(20%)
   Stars = 5⭐(90%+), 4⭐(75%+), 3⭐(60%+), 2⭐(40%+), 1⭐(<40%)
   Bonus/Penalty = +10%, +5%, 0%, -5%, -10%

5. CLAUSE 9 LOGIC
   Contractor Fault   → Part-B deduction (proportional to downtime)
   NGT/Govt Shutdown  → 50:50 share + standby allowance
   MGQ Achieved       → No deduction (fairness rule)

6. PAYMENT HOLD
   Blocks payment if:
   ❌ Agreement unsigned
   ❌ Pending KPI entries
   ❌ Performance < minimum threshold
   ❌ Type-specific validation missing

7. MONTHLY BILLING
   Wizard computes:
   → Part-A + Part-B - Deductions + Bonus/Penalty + Variance - TDS
   → Attaches 4 PDF reports (attendance, diesel, maintenance, breakdown)
   → Creates draft vendor bill
   → Multi-level approval chain
